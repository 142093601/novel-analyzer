# Analysis Framework Reference

## Plot Structure Models

### Three-Act Structure
- Act 1 (Setup): Introduction, inciting incident (~25%)
- Act 2 (Confrontation): Rising action, midpoint, complications (~50%)
- Act 3 (Resolution): Climax, falling action, denouement (~25%)

### Hero's Journey
- Ordinary World → Call to Adventure → Refusal → Mentor → Crossing Threshold
- Tests/Allies/Enemies → Approach → Ordeal → Reward → Return

### Chinese Web Novel常见结构 (Common Chinese Web Novel Structure)
- 开局 (Opening Hook): Power system introduction, initial conflict
- 成长 (Growth Arc): Skill acquisition, rank advancement, faction building
- 高潮 (Climax Arc): Major confrontation, truth reveal
- 结局 (Resolution): World-level conflict resolution

## Character Analysis Dimensions

For each character, extract:
1. **Identity**: Name, aliases, age, role (protagonist/antagonist/supporting)
2. **Motivation**: What drives them? External goals + internal needs
3. **Conflict**: Primary obstacles, internal struggles
4. **Arc**: Starting state → key turning points → ending state
5. **Relationships**: Map to other characters with relationship type + strength

### Relationship Types
- 同盟 (Ally): Cooperative, shared goals
- 对立 (Antagonist): Opposing goals, conflict
- 师徒 (Mentor-Mentee): Teaching, guidance
- 暧昧/爱情 (Romance): Romantic tension or relationship
- 血缘 (Family): Kinship bonds
- 利益 (Transactional): Based on mutual benefit
- 亦敌亦友 (Frenemy): Complex, shifting

## Theme Extraction

Look for:
- Repeated imagery or symbols
- Character dialogues that express philosophical positions
- Structural parallels (mirroring, contrast, repetition)
- Title/subtitle significance
- Resolution patterns (what message does the ending convey?)

## Tension/Pacing Heatmap Criteria

Rate each chapter/section on:
- **Action density**: Events per 1000 words
- **Emotional intensity**: Stakes, danger, revelation
- **Information reveals**: New worldbuilding, plot twists, character reveals
- **Pacing**: Dialogue-heavy (slow) vs action-heavy (fast)

Use 1-5 scale for each dimension.

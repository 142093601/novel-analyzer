# 甜宠 写作技法参考

> 来源：1篇甜宠小说分析。写作阶段使用。

## 开篇钩子

**C. 地狱开局** — 女主处于最差状态（十八线小演员/负债），制造求生欲
- 结构：底层现状 → 天上掉馅饼但有风险 → 被迫接受
- 效果：读者代入底层逆袭的爽感 + 好奇"为什么找她"
- 示例："作为一个出道三年还在演丫鬟甲、路人乙、尸体丙的十八线小演员，三十万是什么概念？"

## 对话模式库

**高冷简短 vs 碎碎念反差：** 男主短句冷淡 + 女主内心疯狂OS
- 例：顾言之："进来。"……苏晚晚内心：近距离看，这张脸真的能杀人。
- 用法：男女主互动的标配，制造反差萌

**自嘲型吐槽：** 女主自嘲 + 内心吐槽 + 行动反差
- 例：干！/ 好家伙，她在人设里居然是个恋爱脑舔狗？/ 苏晚晚嘴角抽了抽
- 用法：制造喜剧效果，增强女主亲和力

**暗糖式温柔：** 男主表面冷淡 + 行动温柔（做早餐/记住喜好/暗中帮忙）
- 例："冰箱里有吃的，随便用。"/ "你就是你，不用演。"
- 用法：不靠甜言蜜语，靠行动撒糖

**脸红心跳描写：** 身体反应 + 心理活动（否认心动）
- 例：苏晚晚的脸爆红……冷静，苏晚晚，你要冷静。这只是演戏。
- 用法：女主心动但不敢承认的阶段

## 爽点实现技法

**反差萌暴击：**
- 压：男主高冷形象（"进来"/"不用演"/"不许喜欢我"）
- 亮：男主做出反差行为（亲自做饭/暗中帮忙/温柔细节）
- 弹：女主心动 + 读者尖叫
- 关键：越冷淡，暴击越猛

**暗恋十年暴击（掉马甲）：**
- 压：女主以为自己是替身/工具人
- 亮：男主说出真相：找了她十年
- 弹：女主震惊感动 + 读者甜到爆炸
- 关键："我会一直找。就算找一辈子，我也会找到你。"

**细节撒糖（暗糖）：**
- 方式：不做大动作，靠微小细节传递深情
- 示例：做早餐、记喜好、冰箱备好食物、暗中帮女主争取角色
- 关键：男主不解释、不邀功，女主自己发现

## 节奏控制参考

| 阶段 | 对话占比 | 句长 | 事件密度 | 情绪基调 |
|------|----------|------|----------|----------|
| 开篇（1章） | 50% | 极短句 | 中 | 紧张+搞笑 |
| 同居（2-3章） | 45% | 短句 | 低 | 甜+暧昧 |
| 掉马甲（4章） | 40% | 中等 | 高 | 震惊+感动 |
| 甜蜜（5-6章） | 50% | 短句 | 低 | 超甜+幸福 |

> 注意：甜宠文对话占比普遍高于其他类型（40-50%），这是正常的。甜宠靠对话和互动传递感情。

## 金句公式

**承诺式：** 你问X → 不可能X → 因为我会一直Y
- 例："你说你找了我十年，那你有没有想过，如果找不到我怎么办？……不可能找不到。因为我会一直找。"
- 用法：男主告白/承诺的高光时刻

**对比总结式：** 最X的事情啊……是Y刚好也Z
- 例："最幸福的事情啊……是我爱的人，刚好也爱我。"
- 用法：结尾升华

**反差简短式：** 简短冷淡的表达 + 含义深远
- 例："你就是你，不用演。""下辈子太远了，我只要这辈子。"
- 用法：男主关键时刻的台词

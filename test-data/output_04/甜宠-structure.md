# 甜宠 剧情模板参考

> 来源：1篇甜宠小说分析（影帝的替身新娘）。大纲阶段使用。

## 核心剧情模板（1种）

### 模板1：合约恋爱变真爱
**一句话：** 被迫当影帝/霸总的合约女友，结果发现对方暗恋自己多年
**适用场景：** 甜宠、合约恋爱、替身反转、暗恋成真
**节拍表：**
1. 女主接受合约：经济困境驱动，被迫当假女友 — 张力 5/10
2. 同居磨合：生活习惯碰撞，暗糖不断（做早餐/送礼物） — 张力 3/10
3. 暗糖升温：男主暗中对女主好，女主心动但不敢确认 — 张力 4/10
4. 掉马甲：发现男主真实感情（暗恋多年/一直在找她） — 张力 7/10
5. 正式在一起：甜蜜日常，双向奔赴 — 张力 2/10
6. 婚后生活：终极甜，收尾温暖 — 张力 2/10

## 冲突升级链

**甜宠标准冲突链：**
```
表层 → 利益 → 价值观 → 情感归属
合约女友身份被误解     经济困境vs金钱诱惑     假装恋爱vs真心相爱     暗恋十年的心酸
```

## 反转技法库

| 反转类型 | 公式 | 出现频率 |
|----------|------|----------|
| 身份反转 | 表面是替身合约女友，实际是男主找了十年的人 | 1/1篇 |
| 动机反转 | 表面是被逼挡桃花，实际是借机接近暗恋对象 | 1/1篇 |

## 爽点节奏公式

**甜宠标准节奏：**
- 开篇（第1章）：小爽点（反差萌：高冷影帝亲自做饭），立人设
- 前期（第2-3章）：小甜点密集（暗糖细节），每章2-3个
- 中期（第4章）：大爽点（掉马甲：暗恋十年暴击），情感爆发
- 后期（第5-6章）：超甜收尾（甜蜜日常+终极告白），连续高位

## 章末钩子技法

| 类型 | 使用频率 | 效果 |
|------|----------|------|
| 情感钩 | 4次 | 甜蜜未完待续的感觉 |
| 悬念钩 | 2次 | 男主反常行为引发好奇 |
| 反转钩 | 1次 | 真相即将揭开 |

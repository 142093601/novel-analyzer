# 重生复仇 剧情模板参考

> 来源：2篇重生复仇小说分析（重生订婚修罗场 + 重生逆袭复仇）。大纲阶段使用。

## 核心剧情模板（2种）

### 模板1：订婚宴修罗场复仇
**一句话：** 重生到订婚宴/婚礼现场，带着前世记忆当场撕破渣男假面
**适用场景：** 重生复仇、渣男打脸、白莲花撕逼
**节拍表：**
1. 前世惨死（被渣男+白莲花害死）→ 重生回关键节点 — 张力 8/10
2. 假装正常，暗中观察所有人，识别敌人和盟友 — 张力 5/10
3. 主动出击：接触关键知情人（律师/老员工），收集证据 — 张力 6/10
4. 发现更深层真相（父亲死因有疑、遗嘱设计），找到盟友 — 张力 7/10
5. 步步为营：职场中压制白莲花，感情上试探渣男 — 张力 6/10
6. 高潮：关键场合（订婚宴/发布会）当众揭露真相 — 张力 9/10
7. 收网：恶人得到惩罚，女主获得新生，与真命天子携手 — 张力 4/10

### 模板2：合约恋爱变真爱
**一句话：** 被迫当影帝/霸总的合约女友，结果发现对方暗恋自己多年
**适用场景：** 甜宠、合约恋爱、替身反转、暗恋成真
**节拍表：**
1. 女主接受合约：经济困境驱动，被迫当假女友 — 张力 5/10
2. 同居磨合：生活习惯碰撞，暗糖不断 — 张力 3/10
3. 暗糖升温：男主暗中对女主好，女主心动但不敢确认 — 张力 4/10
4. 掉马甲：发现男主真实感情（暗恋多年/一直在找她） — 张力 7/10
5. 正式在一起：甜蜜日常，双向奔赴 — 张力 2/10
6. 婚后生活：终极甜，收尾温暖 — 张力 2/10

## 冲突升级链

**重生复仇标准冲突链：**
```
表层 → 利益 → 价值观 → 情感归属
订婚宴上的虚假祝福     林氏集团继承权之争     真心vs利用     被最亲的人背叛
```

## 反转技法库

| 反转类型 | 公式 | 出现频率 |
|----------|------|----------|
| 身份反转 | 女主从被蒙蔽的傻白甜变成重生复仇者 | 2/2篇 |
| 动机反转 | 渣男接近的真实目的被揭穿（为财产不为爱情） | 2/2篇 |
| 立场反转 | 一直以为的慈母/闺蜜实际是幕后推手 | 1/2篇 |

## 爽点节奏公式

**重生复仇标准节奏：**
- 开篇（第1章）：超大憋屈（前世惨死），立即接重生反转
- 前期（第2-3章）：小爽点密集（不信任+试探+收集证据），每章1个
- 中期（第4-5章）：中爽点（职场压制+发现真相），2-3章1个
- 高潮（第5-6章）：大爽点（当众揭露+收网），集中爆发
- 结尾（第7章）：情感收束（婚礼/新生），不以爽收

## 章末钩子技法

| 类型 | 使用频率 | 效果 |
|------|----------|------|
| 悬念钩 | 3次 | 揭示关键信息的一半，引出"接下来怎么办" |
| 反转钩 | 2次 | 章末突然发现新真相 |
| 危机钩 | 1次 | 新威胁出现 |
| 情感钩 | 2次 | 情绪高点截断/温暖收尾 |

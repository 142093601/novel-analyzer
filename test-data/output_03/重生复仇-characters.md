# 重生复仇 角色原型参考

> 来源：2篇重生复仇小说分析。大纲阶段使用。

## 标准角色配置

**重生复仇必备角色（3个）：**
1. **重生复仇女主** — 承载读者代入，执行复仇计划
   - 典型特征：隐忍、果决、擅长伪装、信息优势（重生记忆）
   - 语言风格：短句为主，表面温柔内心狠辣，内心独白用反问
   - 标志性表达："这一世"、"前世的我"、"可惜"、"等着瞧"
   - 出现率：10/10篇重生文

2. **渣男未婚夫** — 制造冲突，提供打脸对象
   - 典型特征：虚伪、贪婪、表面深情、精于算计
   - 语言风格：温柔长句为主，甜言蜜语密集，暴露后恼羞成短句
   - 标志性表达："我会一辈子对你好的"、"不管怎样"
   - 出现率：9/10篇

3. **白莲花女配** — 表面甜实际毒，制造情感冲突
   - 典型特征：表面乖巧、内心恶毒、演技好、野心大
   - 语言风格：甜腻语气（伪装期）→ 尖酸刻薄（暴露后）
   - 标志性表达："姐姐今天真漂亮"、"姐姐放心"
   - 出现率：8/10篇

**重生复仇可选角色（3个）：**
1. **忠犬男主** — 正牌男主，提供情感支持和外部助力（7/10篇）
2. **恶毒继母** — 幕后推手，家族阴谋的执行者（6/10篇）
3. **正义盟友** — 律师/老员工等，提供关键证据（5/10篇）

## 角色语言特征模板

**重生复仇女主说话方式：**
- 句式：短句为主（5-10字），陈述句多，内心独白用反问
- 用词：表面温柔克制，内心狠辣果决
- 口头禅："这一世"、"前世的我"、"可惜，这一世……"
- 典型台词：
  - "这一世，我绝不会让你们得逞。"
  - "林晚，你的演技，能骗我多久？"
  - "我笑了笑，心里却在冷笑。"
  - "上辈子的我听到这话，只会觉得妹妹懂事贴心。可现在……"

**渣男未婚夫说话方式：**
- 句式：温柔长句（假面时期）→ 短句+推卸（暴露后）
- 用词：深情承诺式表达 → 恼羞成怒
- 口头禅："念念"、"我会……"
- 典型台词：
  - "念念，我会一辈子对你好的。"
  - "不管你有没有钱，我都爱你。"

**白莲花女配说话方式：**
- 句式：甜腻+装无辜（伪装期）→ 尖酸+嘲讽（暴露后）
- 用词：关心/羡慕的表面话 → 恶毒的真心话
- 典型台词：
  - "姐姐今天真漂亮，姐夫你可要好好珍惜姐姐啊。"
  - "子墨从来就没爱过你，你知道吗？"

## 角色弧光模板

**复仇觉醒型：**
- 起点：天真善良，被渣男和白莲花蒙蔽
- 转折：被背叛致死，重生回到关键节点
- 过程：觉醒→冷酷布局→在爱与恨间找到平衡
- 终点：强大但不失去人性，获得真正的爱情

**被爱觉醒型：**
- 起点：自卑/不相信自己值得被爱
- 转折：男主的持续温柔行动
- 过程：心动→纠结→确认→敞开心扉
- 终点：自信且被深爱
